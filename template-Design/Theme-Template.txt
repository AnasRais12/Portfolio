Font Size Recommendations
Navbar:

Font Size: 16px - 18px (readable and accessible)
Font Weight: 600 (semi-bold)
Headings:

Main Heading (h1): 32px - 36px
Subheading (h2): 24px - 28px
Section Headings (h3): 20px - 24px
Font Weight:
Main Heading: 700 (bold)
Subheadings: 600 (semi-bold)
Section Headings: 500 (medium)
Body Text:

Font Size: 14px - 16px (comfortable for reading)
Font Weight: 400 (normal)
Buttons:

Font Size: 16px - 18px
Font Weight: 600 (semi-bold)
Padding: 10px 20px (for better clickability)
Additional Tips
Line Height: Aap line height ko 1.5 set kar sakte hain body text ke liye taaki readability behtar ho.
Font Family: Aap sans-serif fonts jaise Roboto, Open Sans, ya Arial choose kar sakte hain, jo modern aur clean dikhte hain.
